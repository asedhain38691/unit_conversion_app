# Unit conversion Flutter App

The Unit Conversion Flutter App lets users convert values between different units of distance and weight. This application supports Kilometers, Miles, Meters, and Feet for distance and support Kilograms, Pounds, Grams, and Ounces for Weight conversion. There are dropdown menus labeled 'From' and 'To' for selecting units and an Input field for entering the value to convert. The result is displayed as [value entered] [from unit] is [converted value] [to unit].
